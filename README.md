# 🏪 Ejercicio 21 – Patrones de Diseño en Java

## 📌 Descripción
Implementación de tres patrones de diseño aplicados
a una tienda de productos.

## 🎯 Patrones practicados
- **Singleton** → Una sola instancia del gestor de tienda
- **Factory** → Fábrica que crea productos físicos y digitales
- **Observer** → Notificaciones automáticas cuando cambia el stock

## 🚀 Cómo ejecutar
1. Clona el repositorio
2. Compila: javac src/**/*.java
3. Ejecuta: java -cp src Main

## 🛠️ Tecnologías
- Java 17
